# Market-Adaptive 7天验证记分表

目标：用 **7 天真实运行样本** 判断系统是否具备进入 **5000 USDT 小资金实盘验证** 的资格。

## 通过线（先看这个）

### 最低生存线
- 高质量 CTA 单：**每周至少 2 笔**
- 高质量单平均净值：**>= 1.2R**
- 低质量单占比：**< 50%**
- 明显手续费型薄单：**持续下降**

### 健康线
- 高质量 CTA 单：**每周 2~3 笔**
- 高质量单平均净值：**>= 1.5R ~ 2.0R**
- 低质量单占比：**< 30%**
- 能出现并拿住 runner 单，而唔係只食到 2~3% move 就走

---

## 分类标准（统一口径）

### 高质量单
满足大部分特征：
- 结构合理，不是尾段追价
- 方向稳定，不是刚开就 signal flip
- 持仓后能走出明显利润展开
- 净值厚度够，唔係手续费型薄利单
- 平仓原因合理（趋势结束 / 分级退出 / ATR stop），唔係无意义噪音打走

### 低质量单
出现任一明显特征：
- 持仓极短（例如 1~3 分钟内结束）
- gross 太薄，net 被手续费吃掉
- 刚开仓就反手 / signal flip
- 入场位置差，明显属于 chasing
- 方向未稳，入场后无 follow-through

### Runner 单
- 持仓明显长于普通单
- 至少跑出 **>= 1.5R** 或明显厚利润
- 退出不是单纯因为轻微噪音

- [ ] 仍不适合 5000 USDT 实盘
- [ ] 可进入极小额试单阶段
- [ ] 可进入 5000 USDT 小资金验证阶段

## 最终结论模板

### 若不通过
结论：
- 高质量单频率不足 / 平均净R不足 / 低质量单仍然过多
- 当前系统仍偏实验阶段，**不建议直接上 5000 USDT 真盘主跑**

下一步优先级：
1. 提升高质量单保留能力
2. 降低低质量单污染率
3. 明确长线单 / 短线单双模管理

### 若勉强通过
结论：
- 系统开始具备正期望可能，但稳定性未完全确认
- 可考虑 **极小额实盘试单**，不建议一下子满额验证

### 若健康通过
结论：
- 系统已初步具备进入 **5000 USDT 小资金实盘验证** 的资格
- 但仍需持续跟踪手续费、滑点、runner 保留率、回撤表现
